# 당일 컨디션 맞춤형 운동 루틴 추천 시스템

## 프로젝트 소개

본 프로젝트는 사용자의 운동 목적, 타겟 부위, 현재 피로도, 운동 가능 시간, 선호 운동 방식을 입력받아 당일 운동 루틴을 추천하는 웹 애플리케이션입니다.

Streamlit 프론트엔드에서 사용자 입력을 받고, FastAPI 백엔드에서 조건 기반 추천 로직을 수행한 뒤 추천 결과를 JSON 형태로 반환합니다. Streamlit은 FastAPI의 응답을 받아 추천 루틴, 운동 강도, 추천 이유, 운동별 세트 수와 반복 횟수, 자세 팁 등을 화면에 출력합니다.

## 기술 스택

* Streamlit
* FastAPI
* Docker
* Docker Compose
* AWS EC2

## 서비스 흐름

사용자 입력 → Streamlit → FastAPI 요청 → 추천 결과 JSON 반환 → Streamlit 화면 출력

## 주요 기능

* 운동 목적 선택
* 타겟 운동 부위 선택
* 현재 피로도 입력
* 운동 가능 시간 입력
* 선호 운동 방식 선택
* 조건 기반 맞춤형 운동 루틴 추천
* 운동 강도 및 루틴 유형 제공
* 운동별 세트 수, 반복 횟수, 휴식 시간 제공
* 운동별 자세 팁 및 참고 검색 링크 제공
* FastAPI에서 생성한 추천 결과를 Streamlit 화면에 출력

## 추천 로직

FastAPI 백엔드는 사용자의 입력값을 바탕으로 운동 루틴을 추천합니다.

추천에는 다음 조건이 반영됩니다.

* 운동 목적
* 타겟 부위
* 현재 피로도
* 운동 가능 시간
* 선호 운동 방식

피로도가 낮고 운동 시간이 충분한 경우에는 비교적 높은 강도의 루틴을 추천하고, 피로도가 높은 경우에는 부상 방지를 위해 머신 또는 저강도 중심의 루틴을 추천합니다. 또한 운동 목적에 따라 반복 횟수와 휴식 시간도 다르게 설정됩니다.

## 주의사항

본 추천 결과는 일반적인 운동 루틴 예시이며, 개인의 건강 상태나 운동 경험에 따라 조절이 필요합니다. 운동 중 통증이나 어지러움이 느껴질 경우 운동을 중단하고 휴식을 권장합니다.
